	# <i class="lni lni-npm" style="position : relative; top : 8px; margin-right : 5px"></i> Installation

    For install **vitest-cucumber** it's easy : 

    ~~~bash
    npm install @amiceli/vitest-cucumber --save
    ~~~

    It's done, you don't need a config file or update *vitest* config.

    ## How it works

    **vitest-cucumber** is like a wrapper. It uses `describe`, `test` etc 
    *vitest* functions.

    Know you can run vitest like you want, for example with coverage : 

    ~~~bash
    vitest --run --coverage
    ~~~
